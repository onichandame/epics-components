# Text Field

A text field for value display. Can be used as an input field. `readValue` must be set to the current reading value whereas `value` should be cleared once the form is submitted.

## Basic

<Preview>
  <Story name="Basic">
    return <Subject label="basic" readValue={100} />
  </Story>
</Preview>

## Changing Value

<Preview>
  <Story name="Changing Value">
    {
      () => {
        const reducer = val => val + 1
        const [value, update] = useReducer(reducer, 0)
        useEffect(() => {
          setInterval(update, 1000)
        }, [])
        return <Subject label="changing value" readValue={value.toString()} />
      }
    }
  </Story>
</Preview>
